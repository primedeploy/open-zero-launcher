# How to contribute?
This is a completely free project that accepts contributions via pull requests on GitHub. This document is responsible for aligning contributions according to the standards established herein. In case of doubts, [open an issue](https://github.com/lfellidev/primedeploy/issues/new).

## Steps
1. Create a fork of this repository.
2. Send your commits in English.
3. Insert a small summary about what was added.
4. Request a pull request.

## Recommendation (optional)
For better semantics in commits, we recommend using the commits mentioned in this open source project. This way it will be easier to evaluate your pull request.
